##汉化约定

1. 请不要使用WINDOWS自带的记事本打开以及保存文件 ∑(っ °Д °;)っ

2. 请保持utf-8编码 ∑(っ °Д °;)っ

3. 提交文件的说明请简单注明提交者名称/日期以及更改的ID段 ∑(っ °Д °;)っ

4. 开工之前，请务必先看看提交记录，请勿重复劳动／人◕ ‿‿ ◕人＼

5. 因为编码问题，不要轻易删除原文句头句尾和命令语句前后的任何空格！以及标点符号！

6. 汉化工作流程：

A ---> B ，所有成员forkB，merge到B

A更新时，对比AB差异，优先汉化A的更新部分

进度一致时，冻结B，fork出稳定版C
